 📊 PCA Project – Dimensionality Reduction

#🚀 Overview

This project implements **Principal Component Analysis (PCA)** to reduce the dimensionality of datasets while preserving maximum variance.


 🎯 Objective

* Simplify high-dimensional data
* Improve visualization
* Enhance model performance

🛠️ Tech Stack

* Python
* NumPy, Pandas
* Matplotlib / Scikit-learn


 ⚙️ Features

* Data preprocessing
* PCA transformation
* Variance analysis
* 2D/3D visualization
